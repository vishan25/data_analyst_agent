Data Analysis API - File Upload Test

API v2.0 Requirements

This interface tests the new API that requires:

  • questions.txt - Required file containing the analysis questions
  • Additional files - Optional files like CSV, images, etc.
  • Synchronous processing - All requests return results immediately
curl "http://localhost:8000/api/" -F "questions_txt=@questions.txt" -F "files=@data.csv"
Required: Must be a .txt file containing your analysis questions (filename must contain 'question')
Optional: Upload additional data files (CSV, images, JSON, etc.)

Testing with curl

You can also test the API directly with curl:

# Multiple file upload (new API v2.0)
curl "http://localhost:8000/api/" -F "questions_txt=@questions.txt" -F "files=@data.csv" -F "files=@image.png"

# Single questions file only
curl "http://localhost:8000/api/" -F "questions_txt=@questions.txt"
